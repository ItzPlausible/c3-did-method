# CoCoA

**Claude Cooperative Assistant** - Personal AI agent system built on Claude Desktop with MCP server integrations.

## Cloudflare Resources
- **Worker:** `cocoa-mvp`
- **D1 Database:** `cocoa-mvp-db`
- **KV:** `cocoa-mvp-COCOA_SESSIONS`
