# Corp2Coop

**Exit-to-Cooperative Platform** - Helping business owners transition to cooperative ownership structures.

## Cloudflare Resources
- **Workers:** `corp2coop`, `corp2coop-api`
- **D1 Database:** `corp2coop-leads`
- **R2 Bucket:** `corp2coop-pdfs`

## Target Market
- UHNW families
- Business owners seeking succession planning
- Pioniro Klanos (Alpha Tranche strategy)

## Local Development
```bash
npm install
npm run dev
```

## Deploy
```bash
npx wrangler deploy
```
